# Exemplos de uso

Transcrições reais do Painel Sintético Concorde rodando dentro do Claude, via conector MCP.
As personas, atributos e fatos citados vêm do painel de verdade — nada aqui é inventado para
a demonstração. Reproduza qualquer um copiando o prompt no seu Claude com o conector instalado
(veja o [README principal](../README.md)).

- [`focus-group-divida.md`](focus-group-divida.md) — focus group de conceito com 3 personas Classe C endividadas
- [`teste-conceito-classe-a.md`](teste-conceito-classe-a.md) — reação de investidor Classe A a uma proposta de benefícios premium
- [`checagem-de-dado.md`](checagem-de-dado.md) — usando o painel como fonte de estatística rastreável (propensão × composição)

> Cada persona só "sabe" o que o segmento dela sustenta estatisticamente. Toda afirmação de número
> nas respostas está ancorada num fato do Grounding, com fonte.
